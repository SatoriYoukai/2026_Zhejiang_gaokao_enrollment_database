浙江志愿落点概率估算器

使用方法
1. 双击“运行_志愿落点概率估算器.bat”。
2. 按提示输入考生位次；直接回车会使用默认位次 39000。
3. 把志愿录入表拖进窗口，或粘贴完整文件路径，然后回车。
4. 如果程序展示“无历史预测位次”，按提示确认或手动输入。
5. 等待 Monte Carlo 模拟结束。

支持的志愿表格式
- .xls
- .xlsx
- .csv

输出位置
结果会写到输入志愿表所在目录：
- *_落点概率估算.xlsx
- *_落点概率估算.csv
- *_落点概率估算报告.txt

注意事项
- 不建议直接双击 exe。直接双击 exe 时，程序结束或报错后窗口可能立刻关闭，看起来像“闪退”。
- 请双击 bat 启动器，它会在结束后暂停窗口，方便查看报错或输出路径。
- 估算结果适合检查志愿表结构，不适合把小数点后的概率当作精确预测。
- 如果杀毒软件提示风险，通常是 PyInstaller 打包程序的常见误报；可以改用源码方式运行。

源码方式运行
在项目目录中执行：
python -X utf8 tools\estimate_volunteer_landing_portable.py "你的志愿表.xlsx" --master data\rank_model_database.csv --user-rank 39000
