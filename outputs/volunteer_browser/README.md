# 浙江 2026 平行志愿筛选器

这是一个离线静态页面，用于筛选浙江省 2026 普通类平行志愿计划，并查看 2023-2025 历史录取分数/位次匹配。

## 使用方式

1. 保持本文件夹内的 `index.html`、`app.js`、`filter_query.js`、`styles.css`、`data.js` 在同一目录。
2. 双击打开 `index.html`。
3. 如果浏览器拦截本地大文件脚本，改用本地服务打开：
   ```powershell
   python -m http.server 8765 --directory outputs/volunteer_browser
   ```
   然后访问 `http://127.0.0.1:8765/`。

## 默认筛选

页面默认筛选：

- 学历：本科
- 选科：物理&化学

可以继续按院校、专业、城市、选科、计划数、2025 位次、最新历史位次、最新历史分数等条件筛选。结果表会直接展示 2026 学费。

## 关键词逻辑

关键词框支持一行可复制的逻辑表达式：

- 词本身表示“包含”，例如 `数学`。
- 空格等价于 `AND`，例如 `数学 统计` 等价于 `数学 AND 统计`。
- 支持 `AND`、`OR`、`NOT`、括号和引号短语。
- 也支持中文写法：`且`、`或`、`非`、`不含`、`包含`。

示例：

```text
数学 AND NOT 师范
(人工智能 OR 智能科学) AND NOT 软件工程
(数学 OR 统计 OR 信息与计算科学) AND NOT (师范 OR 金融数学 OR 经济统计)
"计算机类" AND NOT 中外合作
```

## 数据说明

- 数据来自 `outputs/clean_database/volunteer_master_2026_with_history.csv`。
- 历史匹配不是官方 2026 专业代码映射，而是基于清洗规则把 2023-2025 历史录取行拼回 2026 志愿项。
- 进入最终志愿单前，应回查详情里的 PDF 页码和官方招生计划。

## 文件说明

- `index.html`: 页面入口。
- `data.js`: 前端数据，约 34MB。
- `filter_query.js`: 关键词逻辑表达式解析器。
- `app.js`: 筛选和表格逻辑。
- `styles.css`: 页面样式。
- `data.json`: 同一份数据的 JSON 版，供二次开发使用。
