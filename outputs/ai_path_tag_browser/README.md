# AI 路径候选标签浏览器（半成品）

这个目录是一套面向“AI 科研路径”选校筛选的静态标签浏览器。它可以直接用浏览器打开，也可以用本地 HTTP 服务打开：

```powershell
python -m http.server 8771 --directory outputs/ai_path_tag_browser
```

然后访问 `http://127.0.0.1:8771/`。

## 状态说明

这是一个研究辅助工具，已经停止继续开发。它保留在仓库中，主要用于复盘和二次开发参考。

它比较有用的部分：

- 展示候选志愿的院校、专业、位次、学费、计划数等结构化信息；
- 用标签和权重浏览候选池；
- 展示部分强标签、弱标签、证据摘要和数据缺口；
- 支持偏好标记和导出配置。

它的明显限制：

- 标签和证据来自一轮实际填报研究，带有个人偏好；
- agent 自动爬取的培养方案、宿舍、保研、氛围等证据缺口较大；
- “证据缺失”不能直接等价于“学校/专业差”；
- 不建议把它当作自动决策系统使用。

## 文件

- `index.html`: 入口页面。
- `app.js`: 前端交互逻辑。
- `styles.css`: 样式。
- `data.json`: 标签浏览器数据。
- `data.js`: 浏览器直接加载用的数据包装文件。
- `current_pool_brief.csv`: 当前候选池简表。

## 生成脚本

生成脚本在：

```text
tools/build_ai_path_tag_browser.py
```

生成过程依赖本仓库中此前筛选、打标、证据汇总产生的中间文件。若要把它改造成通用产品，建议先重做数据契约，拆掉个人化偏好和本次填报专用标签。
